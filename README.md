# 1. 프로젝트 개요
jsp를 기반으로 작업한 커뮤니티 사이트<br><br>
회원가입, 로그인, 게시판, 댓글 기능을 지원하며, Bootstrap을 이용한 반응형 웹페이지
<br><br>

- JSP
- Javascript
- Bootstrap
- MariaDB
